Movie API Documentation

Request URL Response
GET /movies Returns a list of all movies
GET /movies/:title Returns data about a single movie by title
GET /genres/:name Returns data about a genre by name/title
GET /directors/:name Returns data about a director by name
POST /users Registers a new user
PUT /users/:username Updates user info
POST /users/:username/movies/:movieID Adds a movie to the user's list of favorites
DELETE /users/:username/movies/:movieID Removes a movie from the user's list of favorites
DELETE /users/:username Deregisters a user
